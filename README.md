## DEMO

![](https://raw.githubusercontent.com/akameco/npm-pixiv/master/media/demo.gif)

## Installation

```
$ npm install --global npm-pixiv
```

## Usage

```
$ npm-pixiv install -S react react-dom
```

## Recommend

Add this to ~/.bashrc or ~/.zshrc:

```
alias npm=npm-pixiv
```
